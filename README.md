# THYRO CARE

A modern diagnostic laboratory management system built using React, TypeScript, Node.js, Express, Supabase, and Tailwind CSS.

## Features
- Patient Registration
- Online Test Booking
- Admin Dashboard
- Test Reports
- Authentication
- Profile Management
- Contact & Appointment
- Responsive Design

## Tech Stack

Frontend
- React
- TypeScript
- Tailwind CSS
- Vite

Backend
- Node.js
- Express.js
- Supabase
- PostgreSQL

## Installation

```bash
npm install
npm run dev
